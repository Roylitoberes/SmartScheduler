# SmartScheduler

SMARTSCHEDULER is a .NET MAUI cross-platform scheduling application designed to help users organize tasks, manage time effectively, and improve productivity with a simple and user-friendly interface.

---

## 📱 Download APK

### ⚠️ Important: You must be logged into GitHub to download

[![Download APK](https://img.shields.io/badge/Download-APK-green?style=for-the-badge&logo=android)](https://github.com/Roylitoberes/SmartScheduler/actions/runs/24574476959)

### Instructions:
1. **Click the button above** or [this link](https://github.com/Roylitoberes/SmartScheduler/actions/runs/24574476959)
2. Scroll down to **"Artifacts"** section
3. Click **"SmartScheduler-APK"** to download
4. **Enable "Unknown Sources"** on your Android device if prompted
5. Open the downloaded APK file to install

### System Requirements:
- Android 5.0 (API 21) or higher
- Minimum 200MB free storage space

### Need help?
If you encounter a "parsing error", make sure:
- You're running Android 5.0 or higher
- You've uninstalled any previous version first
- You've enabled installation from unknown sources
